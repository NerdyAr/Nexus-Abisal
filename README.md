# Nexus Abisal

**Nexus Abisal** es un blog de inteligencia estratégica que explora las intersecciones entre análisis forense histórico, marcos de inteligencia como 5C/2I, laboratorios lógicos y dinámicas estáticas en movimiento. Inspirado en una estética de "consola de inteligencia" minimalista, el sitio se despliega como un sitio web estático en GitHub Pages, fomentando la colaboración open-source y la transparencia analítica.

## Descripción General
- **Tecnologías**: HTML5, CSS3 (Grid/Flexbox), JavaScript vanilla. Fuentes: Playfair Display (títulos) y JetBrains Mono (cuerpo). Paleta: Azul Abismo (#0D1B2A), Gris Perla (#E0E1DD), Cobre (#B07D62).
- **Estructura del Sitio**:
  - `index.html`: Página principal con hero, manifiesto animado y rejilla de posts.
  - `post*.html`: Páginas individuales de artículos (ej. `post1.html`).
  - Estilos y scripts incrustados para simplicidad y rendimiento.
- **Interactividad**: Cursor personalizado con rastro de "hilo de cobre", comentarios vía Utterances (GitHub Issues) y formulario de email moderado.
- **Despliegue**: GitHub Pages. Clona este repositorio y accede a `/docs` o la rama principal para visualizar.

## Manifiesto de Nexus Abisal

### Fin
Explorar la intersección entre inteligencia estratégica y narrativas creativas para desentrañar patrones ocultos en la historia, la tecnología y las redes abisales de conocimiento, fomentando una comprensión profunda y conectada de los sistemas complejos.

### Misión
Proporcionar análisis rigurosos, accesibles y multidisciplinarios que empoderen a profesionales, investigadores y curiosos en la toma de decisiones informadas, democratizando herramientas de inteligencia open-source.

### Visión
Construir un ecosistema global donde la inteligencia estratégica trascienda fronteras disciplinares, impulsado por colaboraciones colectivas y perspectivas innovadoras que revelen nexos invisibles en un mundo interconectado.

### Valores
- **Transparencia**: Todo análisis es reproducible y documentado.
- **Precisión Analítica**: Basada en evidencia forense y lógica deductiva.
- **Colaboración Abierta**: Invitación a contribuciones vía GitHub Issues y pull requests.
- **Respeto Intelectual**: Diversidad de voces, sin sesgos ideológicos, priorizando el valor edificante.

### Objetivos Operativos (Tácticos)
- Publicar al menos un artículo semanal en las secciones temáticas ([01] Historical Forensics, [02] 5C/2I Intel, [03] The Logic Lab, [04] Static in Motion).
- Optimizar el SEO y accesibilidad (responsivo, alt-text en elementos visuales) para alcanzar 1.000 visitas mensuales en los primeros seis meses.
- Fomentar interacciones comunitarias mediante Utterances y moderación de comentarios (excluyendo contenido no edificante como insultos o sesgos divisivos).
- Mantener actualizaciones mensuales del código para micro-animaciones y compatibilidad con navegadores modernos.

### Objetivos No Operativos (Estratégicos)
- Establecer alianzas con redes académicas y profesionales para coautorías en análisis forense histórico-digital.
- Desarrollar benchmarks en modelos lógicos para escenarios de inteligencia estática, publicando recursos open-source descargables.
- Expandir a una red global de colaboradores, midiendo impacto mediante métricas de engagement (ej. forks y estrellas en GitHub).
- Contribuir al legado de la inteligencia estratégica colectiva, reconociendo el rol de IAs cooperativas y comunidades humanas en la generación de insights.

## Instrucciones de Contribución
1. **Fork** el repositorio.
2. Cree una rama: `git checkout -b feature/nueva-seccion`.
3. Realice cambios y commitee: `git commit -m "Agrega análisis forense"`.
4. Pushee a la rama y cree un Pull Request.
5. Discuta cambios en Issues. Todos los comentarios se moderan para mantener un entorno respetuoso.

## Configuración de Utterances
Para habilitar comentarios:
1. Cree un repositorio separado para issues (ej. "nexus-abisal-comments").
2. En `index.html` y posts, actualice el script: `<script src="https://utteranc.es/client.js" repo="tu-usuario/nexus-abisal-comments" ...></script>`.
